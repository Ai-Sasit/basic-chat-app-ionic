# basic-chat-app-ionic

## Getting Start
### 1. setup backend & create database
#### 1.1 install dependencies
```
cd basic-chat-app-ionic/backend
yarn install
```
#### 1.2 create `MongoDB` via Docker:

```
docker-compose up -d
```
#### 1.3 run backend
```
yarn dev
```

### 2. setup frontend
#### 2.1 install dependencies
```
yarn install
```
#### 2.2 run app
```
ionic serve
```
